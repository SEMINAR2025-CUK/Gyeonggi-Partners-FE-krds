# 경기 파트너스 프로젝트 - FE

본 프로젝트는 krds 디자인 패턴을 준수하기 위해서 krds-react에서 제공해주는 개발 템플릿 위에서 개발작업이 진행되었습니다.
Fork 날짜는 2025년 9월 30일 이후임을 명백히 공지합니다.

# KRDS-React

KRDS-React는 한국 정부 디지털 서비스 디자인 시스템을 기반으로 한 React 컴포넌트 라이브러리입니다.

## 특징

- 한국 정부 디지털 서비스 디자인 가이드라인 준수
- React 기반의 재사용 가능한 UI 컴포넌트
- TypeScript 지원

## 설치

```bash
pnpm install
```

### 스크립트

- `pnpm dev`: 개발 서버 실행
- `pnpm build`: 프로덕션용 빌드 생성

## 기술 스택

- React
- TypeScript
- Vite
- tailwind

## 활용 핵심 라이브러리
- krds-tailwind
- krds-core
